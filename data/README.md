# Data

This project uses the public **Amazon Products Dataset 2023 (~1.4M products)** from Kaggle.
The large product file `amazon_products.csv` (~358 MB) is **not** redistributed here — download it and load it as `amazon_products`:

**https://www.kaggle.com/datasets/asaniczka/amazon-products-dataset-2023-1-4m-products**

`amazon_categories.csv` (the small category-id → name lookup, included in this folder) is used to label the predicted products.
